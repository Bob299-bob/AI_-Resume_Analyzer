# 🚀 AI Resume Analyzer

An intelligent **AI-powered Resume Analyzer** that evaluates your resume against a given Job Description (JD) and provides instant insights like ATS score, skill match, missing skills, and career recommendations.

---

## ✨ Features

✔ ATS Score (Resume vs Job Description Match %)  
✔ Smart Job Role Prediction  
✔ Matched Skills Extraction  
✔ Missing Skills Identification  
✔ Skill Gap Analysis  
✔ Career Domain Detection  
✔ Visual Skill Comparison Chart  
✔ Downloadable PDF Report  
✔ Simple and interactive Streamlit UI  

---

## 🧠 How It Works

1. Upload your resume (PDF)
2. Paste a Job Description
3. System processes text using NLP
4. Sentence Transformers generate embeddings
5. Cosine similarity calculates ATS score
6. Skills are extracted and compared
7. Results are displayed with charts and reports
